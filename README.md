# a.ndr.ooo Nix config

This is my Nix configuration! It defines config for:
- kiwi: my laptop
- lyoko: my desktop
- xana: my tiny home server
- replika: the raspberry pi attached to my 3D printer

This repo uses flakes and home-manager. I'm using agenix to manage secrets and nix-colors for theming. I primarily use [sway](https://swaywm.org/) as a Wayland compositor but I hop around from time to time.

The core setup is based on [Misterio77/nix-starter-configs](https://github.com/Misterio77/nix-starter-configs).
